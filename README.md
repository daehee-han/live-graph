# live-graph

live sensor graph with Flask+SocketIO

## Support

- Windows

- MacOS X

- Linux

## Usage in Linux

### Python 2.7

#### Installing packages

```bash
$ make
```

#### Running server

```bash
$ make start
```

### Python 3.x

#### Installing packages

```bash
$ sudo pip install -r requirements
```

#### Running server

```bash
$ python server.py
```
